Download

Compiled binaries

The easiest way to obtain the program is download compiled binaries for Linux and Mac platforms. Just download the archive, unpack it and follow the instruction inside. You don't need to install the binaries, therefore you can use it without root permission. Please note, that output packing (-a option) is not available in these binaries.

Source code

Building the program from source code is suggested for advanced users, who need output packing or want to change the program code. I can reccomend to use Ubuntu 12.04 or later for compiling the code. You can also install compiled program to your system, if you have an administrator rights. The program can be compiled for Windows platfrom also, but I've never try this. Tutorial and manual sources are also available.
